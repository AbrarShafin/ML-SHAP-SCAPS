# ML–SHAP–SCAPS Framework (Code & Data Repository)

## 📌 Overview
This repository provides the dataset, source codes, and generated figures used in the study:

"Machine Learning-Aided Modelling and Absorption Optimization in Stacked-Absorber Heterostructure Solar Cells"

The purpose of this repository is to ensure transparency and reproducibility of the proposed ML–SHAP–SCAPS framework.

---

## 📂 Contents

- **Dataset/**  
  Contains all simulation datasets (Excel/CSV) used for training and evaluation.

- **ML_codes/**  
  Includes Python scripts for:
  - Machine learning model development
  - Model evaluation
  - SHAP-based explainability analysis

- **figures/**  
  Contains plots and visual outputs used in the manuscript.

---

## ⚠️ Note on Execution
The codes are provided for **research transparency and reference purposes**.

Due to environment-specific configurations (e.g., SCAPS simulation setup, file paths, and dependencies), the scripts may require modification before execution.

---

## 🔍 Method Summary
- SCAPS-1D simulations used for data generation  
- Machine learning models applied for prediction  
- SHAP used for interpretability
